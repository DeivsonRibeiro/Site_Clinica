# 🏥 Site Clínica

## 📌 Sobre o Projeto

Projeto de um site institucional para clínica médica, com foco em apresentação de serviços, contato e interface amigável para o usuário.

## 🛠️ Tecnologias Utilizadas

* HTML
* CSS
* JavaScript

## ⚙️ Funcionalidades

* Página inicial informativa
* Seção de serviços
* Contato
* Layout responsivo

## 🎯 Objetivo

Simular um projeto real de site institucional, aplicando conceitos de design e desenvolvimento front-end.

## 📚 Aprendizados

* Estruturação de páginas
* Responsividade
* Organização visual
* UX básica

## ▶️ Como Executar

```bash
git clone https://github.com/DeivsonRibeiro/Site_Clinica.git
cd Site_Clinica
```

Abra o arquivo `index.html` no navegador.
